## 1. 产品概述

透析患者健康追踪应用，帮助透析患者严格控制每日摄水量、超滤量、水果摄入量及钾元素摄入。面向需要进行血液透析/腹膜透析治疗的患者群体，通过可视化数据帮助患者遵循医嘱，预防高钾血症和心衰等并发症。

- 主要目的：让透析患者能够便捷记录每日健康指标，并对超标情况及时预警
- 目标用户：血液透析或腹膜透析患者，需控制水、钾摄入
- 市场价值：填补国内缺乏针对透析患者的轻量化、隐私优先（数据本地存储）的健康追踪工具空白

## 2. 核心功能

### 2.1 用户角色

无角色区分，单用户本地使用模式。所有数据存储在浏览器 localStorage 中，无需注册登录，注重隐私保护。

### 2.2 功能模块

1. **今日仪表盘**：当日指标总览、快速记录、超标预警、当日剩余配额
2. **记录中心**：历史记录列表、趋势图表、按类型筛选、每日详情
3. **水果钾含量库**：常见水果钾含量参考、搜索、自定义添加新水果

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 今日仪表盘 | 指标卡片区 | 显示当日摄水量、超滤量、水果摄入量、钾摄入量，配额进度条与剩余量，超标红色预警 |
| 今日仪表盘 | 快速记录区 | 三个快速记录卡片（饮水/超滤/水果），点击展开表单录入数值或选择水果 |
| 今日仪表盘 | 时段分布图 | 当日饮水时间分布柱状图，显示饮水节奏 |

## 3. 核心流程

### 3.1 用户主流程

1. 用户首次打开应用 → 设置每日限额（摄水量、钾摄入量等）
2. 进入今日仪表盘 → 查看当日已摄入量与剩余配额
3. 点击"饮水"快速记录 → 输入毫升数 → 进度条更新，若超标显示红色预警
4. 点击"水果"快速记录 → 选择水果 + 输入重量 → 自动计算钾摄入量，累加至当日
5. 透析后点击"超滤"快速记录 → 输入超滤量 → 记入当日数据
6. 切换至记录中心 → 查看历史趋势，评估饮食执行情况
7. 查询水果钾含量库 → 了解新水果是否适合食用

## 4. 用户界面设计

### 4.1 设计风格

**美学方向：禅意医疗治愈风（Zen Medical Healing）**

- **主色调**：米白底色 (#F8F5F0) + 鼠尾草绿 (#7A9B7E) 深青主色 (#2D5F5D)
- **辅助色**：陶土橙 (#D97757) 用于预警，柔米色 (#E8E0D5) 用于卡片背景
- **按钮风格**：圆角矩形 (12px)，主按钮采用深青底色 + 微妙阴影；次按钮采用描边样式
- **字体**：标题用 "Noto Serif SC"（思源宋体）传递专业可靠感；正文用 "Noto Sans SC"（思源黑体）
- **布局风格**：卡片式布局 + 顶部导航 + 大量留白；桌面端采用三栏式仪表盘布局
- **图标风格**：线性图标 (Lucide) 配合柔和颜色，避免冷硬的医疗感
- **动画**：进度条平滑过渡 (cubic-bezier)、卡片悬停微抬升、数字滚动效果
- **质感细节**：背景使用细微噪点纹理 + 柔和渐变，营造温暖治愈氛围

### 4.2 页面设计概览

### 4.3 响应式设计

- 桌面优先（1280px+），三栏式仪表盘布局
- 平板（768-1280px），两栏布局
- 移动端（<768px），单栏垂直堆叠，导航栏改为底部 Tab Bar
- 触摸优化：按钮最小点击区域 44x44px，表单输入框增大

### 4.4 3D 场景指引

不适用。本项目为纯 2D 数据可视化应用。
